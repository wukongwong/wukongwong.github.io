2024

  1. IJHR
    Seeing is Grasping: A Novel Arm-Gripper Grasping Synergy Framework without any Calibration
    Shuaijun Wang, Lining Sun, Qiang Li, and 2 more authors
    International Journal of Humanoid Robotics, 2024
    SCI
    Abs DOI Bib HTML Video

    This work proposes an end-to-end visual servoing framework for adaptive arm-hand grasping that requires no camera calibration or hand-eye transformation. We demonstrate real-time dynamic 3D grasping across diverse objects using only low-cost RGB cameras.

2023

  1. Front. Neurorobot.
    Learning Adaptive Reaching and Pushing Skills Using Contact Information
    Shuaijun Wang, Lining Sun, Fusheng Zha, and 2 more authors
    Frontiers in Neurorobotics, 2023
    SCI Q2
    Abs DOI Bib HTML Video

    We develop a reinforcement learning-based pushing strategy using Soft Actor-Critic (SAC) with explicit contact modeling. The approach demonstrates improved generalization and training efficiency for contact-aware object manipulation tasks.

2022

  1. TMECH
    Learning Adaptive Grasping From Human Demonstrations
    Shuaijun Wang, Lining Sun, Alex Li, and 2 more authors
    IEEE/ASME Transactions on Mechatronics, 2022
    SCI Q1
    Abs DOI Bib HTML Video

    This paper presents a learning-based approach to enable adaptive grasping from human demonstrations. We collect human teleoperation data and use supervised learning to train neural networks that can handle object variability in size and stiffness. The proposed method demonstrates robust performance across different object categories.
